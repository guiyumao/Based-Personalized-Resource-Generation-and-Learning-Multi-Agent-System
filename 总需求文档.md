# 软件杯比赛项目：个性化学习智能体系统 - 设计方案

## 一、赛题核心要求回顾

- **对话式学习画像自主构建**：自然语言对话抽取 ≥6 个维度，动态更新。
- **多智能体协同的资源生成**：至少生成 5 种类型个性化资源（文档、思维导图、题目、拓展阅读、代码案例/视频等）。
- **个性化学习路径规划 + 资源推送**：基于画像和进度，自动规划学习步骤，精准推送资源。
- **用户画像构建**：基于知识基础，认知风格，易错点偏好，学习节奏，兴趣方向，学习目标。。
- **智能辅导（加分）**：多模态答疑（文字、图解、短视频）。
- **学习效果评估（加分）**：多维度评估，动态调整策略。
## 二、整体技术选型

| 层次 | 技术 | 说明 |
|------|------|------|
| 前端 | Vue3 + Element Plus + ECharts | 流式输出（SSE）、Markdown 渲染、卡片化展示 |
| 后端 | Spring Boot + MyBatis-Plus | 统一 Java 技术栈，减少跨语言调用 |
| 数据库 | PostgreSQL + pgvector | 主数据 + 向量检索（RAG） |
| 缓存 | Redis | 热点数据、Token、部分画像缓存 |
| 消息队列 | RabbitMQ（可选） | 异步处理耗时生成任务 |
| AI 服务 | DeepSeek API（付费版） | 大模型推理 + Embedding |
| 容器化 | Docker | 部署与演示 |

## 三、智能体设计（核心 4 个 + 加分 2 个）

### 必须实现的智能体（4 个逻辑体）

| 智能体 | 职责 | 对应 API 端点 |
|--------|------|---------------|
| 画像构建智能体 | 对话抽取 6 维度，随学随新 | `POST /api/profile/chat` |
| 资源生成协调智能体 | 拆解用户需求，调度下游生成 | `POST /api/resource/generate` |
| 通用资源生成智能体 | 根据类型参数生成文档/思维导图/题目/代码案例/拓展阅读 | 由协调智能体调用 |
| 学习路径规划 + 推送智能体 | 生成学习步骤，推送资源 | `GET /api/learning-path`<br>`GET /api/learning-path/resources/{id}` |

> 注：通用资源生成智能体可支持 5 种以上资源类型，通过枚举参数 + 不同 Prompt 实现，无需拆分为多个独立代码类。

### 加分项智能体（可选）

| 智能体 | 职责 |
|--------|------|
| 智能辅导智能体 | 基于 RAG 检索知识库，多模态答疑 |
| 学习效果评估智能体 | 分析行为数据，给出评估报告并触发路径调整 |

## 四、系统模块与页面结构

### 学生端（核心）

- 登录 / 注册 / 个人中心（JWT + BCrypt）
- 首页仪表盘
    - 今日学习任务（智能体生成）
    - 继续学习（记录上次位置）
    - 资源推荐（基于画像）
- 智能学习空间
    - 对话式画像构建
    - 资源生成中心
    - 学习路径视图
- 学业智析（学习分析平台）
    - 六维度雷达图
    - 学习效果评估曲线
    - AI 学习建议
- 智能辅导员（加分项）

### 管理员端（极简）

- 上传课程资料（触发向量化入库）

> 教练端 / 教师端 **已裁剪**，节省工作量。

## 五、接口完整清单（30 个）

> **登录策略**：首页公开接口无需登录；点击需要登录的功能时前端检查 Token，无则弹窗登录。

### 1. 用户认证模块（4 个，公开）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 用户注册 | POST | `/api/auth/register` | 手机/邮箱 + 密码 |
| 用户登录 | POST | `/api/auth/login` | 返回 JWT + 基本信息 |
| 刷新 Token | POST | `/api/auth/refresh` | 可选 |
| 登出 | POST | `/api/auth/logout` | 可选黑名单 |

### 2. 首页模块（3 个，部分公开）

| 接口 | 方法 | 路径 | 登录要求 | 说明 |
|------|------|------|----------|------|
| 获取推荐资料列表 | GET | `/api/home/recommendations` | 否 | 冷启动通用推荐；登录后基于画像 |
| 获取学习进度卡片 | GET | `/api/home/progress` | 是 | 上次学习的资源 ID 和位置 |
| 获取今日学习任务 | GET | `/api/home/tasks` | 是 | 由路径规划智能体生成 |

### 3. 画像构建模块（3 个，需登录）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 发送对话消息 | POST | `/api/profile/chat` | 调用画像智能体，返回回复 + 更新维度 |
| 获取当前画像 | GET | `/api/profile` | 返回六维度数据（雷达图） |
| 手动修正画像 | PUT | `/api/profile` | 可选，允许用户微调 |

### 4. 资源生成模块（3 个，需登录）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 请求生成资源 | POST | `/api/resource/generate` | 同步返回（2-5 秒），或返回任务 ID |
| 流式生成（SSE） | GET | `/api/resource/generate/stream` | 可选，提升体验 |
| 获取生成历史 | GET | `/api/resource/history` | 分页返回历史资源 |

### 5. 资源内容访问模块（5 个，需登录）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 获取资源详情 | GET | `/api/resource/{resourceId}` | 返回完整内容，自动记录进度 |
| 获取题目列表 | GET | `/api/resource/questions/{resourceId}` | 仅供练习使用 |
| 提交答案（判题） | POST | `/api/resource/answers` | 支持选择/填空，编程题调用沙箱 |
| 上报学习进度 | POST | `/api/resource/progress` | 前端定时上报当前位置 |
| 获取资源历史进度 | GET | `/api/resource/progress/{resourceId}` | 用于“继续学习”恢复 |

### 6. 学习路径与推送模块（3 个，需登录）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 获取学习路径 | GET | `/api/learning-path` | 返回节点列表（顺序 + 预估时间） |
| 手动调整路径 | POST | `/api/learning-path/adjust` | 用户标记跳过/已学会 |
| 获取知识点资源 | GET | `/api/learning-path/resources/{knowledgeId}` | 返回推送的文档/视频/题库 |

### 7. 学业分析模块（3 个，需登录）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 获取评估报告 | GET | `/api/analytics/report` | 正确率趋势、时长分布等 |
| 获取雷达图数据 | GET | `/api/analytics/radar` | 六维度数值 |
| 获取 AI 学习建议 | GET | `/api/analytics/suggestion` | 基于效果评估智能体生成 |

### 8. 个人中心模块（3 个，需登录）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 获取个人信息 | GET | `/api/user/profile` | 昵称、头像、专业等 |
| 更新个人信息 | PUT | `/api/user/profile` |  |
| 修改密码 | PUT | `/api/user/password` | 需验证旧密码 |

### 9. 管理员上传模块（2 个，管理员权限）

## 六、开发优先级建议

### 第一优先级（跑通核心流程，可演示）
- 用户认证（登录即可，注册可后做）
- 首页推荐（先硬编码几条演示数据）
- 资源生成（最少 1 种类型，如讲解文档）
- 资源内容访问（查看文档 + 记录进度）
- 画像构建（最简单的对话抽取，可先存到数据库）

### 第二优先级（完善体验）
- 学习路径规划（基于画像简单推荐顺序）
- 学业分析（雷达图展示画像维度）
- 做题 + 判题（选择题/填空题）

### 第三优先级（加分亮点）
- 智能辅导员（RAG）
- 流式生成（SSE）
- 上传资料接口（管理员）

## 七、防幻觉与内容安全

- 各智能体采用不同 `temperature` 参数（文档生成 0.5，题目生成 0.3 并强制 JSON Mode）。
- 在 Prompt 中明确要求“若信息不确定，请回答‘不确定’并建议查阅人工资料”。
- 后端增加敏感词过滤（可调用简单字典或讯飞内容安全 API）。

## 八、成本估算（DeepSeek API）

- **输出价格**：6 元 / 百万 Token
- **缓存命中输入**：0.025 元 / 百万 Token
- 单个知识点生成全部资源（文档+思维导图+题目）：约 6k–10k Token → **约 0.036–0.06 元**
- 支持几百学生日常使用，总成本可忽略。
