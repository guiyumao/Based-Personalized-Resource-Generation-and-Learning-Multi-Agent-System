🎨 设计思路

🔀 导航分离

将原本挤在一个页面的 15+ 功能面板按业务拆分为独立页面， 通过侧边栏菜单串联。用户不再需要"滚屏找功能"。

✨ 科技感视觉

深色主题 + 青蓝/橙暖双强调色，磨砂玻璃面板， 渐变高光线条，脉冲动画提示。有未来感但不花哨。

📱 响应式适配

侧边栏可在移动端自动折叠为图标模式， 统计网格和内容区域自适应列数变化。

🎯 渐进迁移

侧边栏路由结构与现有 Vue Router 可直接对应。 每个"页面"对应一个独立 .vue 组件， 按功能模块逐步拆分，无需一次性重写。

🌊 动态背景

登录页使用 Vanta.js WAVES 效果，深色波涛动态背景 配合磨砂玻璃卡片，比纯静态渐变更显精致和专业感。

🧩 组件化改造

建议后续将可复用 UI 提取为独立组件（统计卡片、 时间线节点、课件卡片、错题卡等），减少重复代码。
