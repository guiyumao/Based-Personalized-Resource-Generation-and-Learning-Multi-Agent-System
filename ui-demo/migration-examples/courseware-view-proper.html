第一类换元法（凑微分法）

第一类换元法的核心思想是识别被积函数中是否出现了某个函数及其导数的乘积。 如果被积函数可以写成 f(g(x)) · g'(x) 的形式，设 u = g(x)，则： 

∫ f(g(x)) · g'(x) dx = ∫ f(u) du

识别技巧：

  1. 观察被积函数中是否有某个表达式及其导数的乘积
  2. 如果 g'(x) 恰好出现（可能差一个常数因子），优先尝试凑微分
  3. 凑出 du = g'(x)dx 后，原积分化为 ∫ f(u) du
  4. 对 u 积分后，将 u = g(x) 代回

典型例题：

求 ∫ 2x · e^(x²) dx

解：注意到 (x²)' = 2x，出现在被积函数中
    令 u = x²，则 du = 2x dx
    原式 = ∫ e^u du = e^u + C = e^(x²) + C

常数因子的处理：如果 g'(x) 差一个常数因子，可以通过乘除常数来调整。 例如：∫ x · sin(x²) dx，这里 (x²)' = 2x， 需要补一个 1/2 来凑微分。
