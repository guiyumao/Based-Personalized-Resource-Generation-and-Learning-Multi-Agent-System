八大智能学习模块

从学习路径规划到课后自测，从知识图谱到智能问答 —— AI 多智能体系统为每个学习环节提供精准支持

🗺️

个性化学习路径

AI 根据掌握度与学习风格，自动生成分阶段学习路径。每阶段包含明确目标与任务清单，动态调整难度。

[ 动图：学习路径时间线展开动画 ]
📚

智能课件生成

基于 RAG 检索增强生成，整合多源知识产出结构化课件。支持难度分级与学习风格适配，可生成多个变体版本。

[ 动图：课件从生成到展示的完整流程 ]
✏️

自适应练习测评

根据课件内容自动生成分层习题，提交后即时反馈与解析。支持单选、填空、简答等多种题型。

[ 动图：答题 → 即时反馈 → 解析展示 ]
📝

智能错题本

自动收录错题，AI 分析错误根因。一键生成同类变式题巩固训练，直至彻底掌握薄弱知识点。

[ 动图：错题重练 → 变式题生成 → 正确通过 ]
🔗

知识图谱可视化

构建学科知识依赖网络，直观展示知识点间的先修关系。节点颜色反映掌握程度，一眼定位薄弱环节。

[ 图片：知识图谱网络交互视图 ]
💬

多智能体问答

不同于通用 ChatBot，系统结合学习画像进行深度分析：识别知识差距、更新学习路线、给出针对性建议。

[ 动图：问答对话 → 结构化分析卡片展开 ]
📈

阶段学习报告

自动汇总阶段学习数据，输出多维度诊断报告。包括正确率趋势、知识覆盖度、薄弱点热力图等。

[ 图片：学习报告仪表盘概览 ]
👥

多角色协同

学生自学 + 教师班级管理 + 管理员系统配置。教师可布置作业、查看学情、批改反馈，形成教学闭环。

AI 技术

多智能体驱动的智能学习引擎

系统采用多智能体协同架构，各 Agent 分工协作，覆盖资源生成、学习诊断、路径规划的完整链路

🧠

多智能体协同架构

资源生成 Agent、学习诊断 Agent、路径规划 Agent 等多个专业智能体分工协作， 通过消息队列（RabbitMQ）异步通信，高效完成复杂学习任务。

LangGraph Multi-Agent RabbitMQ
🔍

RAG 检索增强生成

课件和习题生成前，先从向量知识库（ChromaDB）检索相关知识片段， 结合 Elasticsearch 全文检索，确保生成内容准确、有据可依。

ChromaDB Elasticsearch Embedding
🕸️

知识图谱推理

基于 Neo4j 图数据库构建学科知识图谱，支持先修依赖推理和知识缺口检测。 掌握度通过图传播算法更新，精准定位薄弱知识节点。

Neo4j Graph Reasoning vis-network
🎯

学习画像与自适应

持续追踪学习行为数据，构建多维度学习画像（掌握度、学习风格、认知能力）。 基于画像动态调整内容难度、呈现方式和推荐策略。
